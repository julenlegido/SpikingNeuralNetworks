**Towards energy-efficient brain-inspired AI: An overview and evaluation of Spiking Neural Networks**

Spiking Neural Networks (SNNs), a paradigm in Artificial Intelligence (AI) that gets inspiration from the physiology of the human brain, offer a compelling alternative to traditional Artificial Neural Networks (ANNs) for realizing Machine Learning (ML) models in an energy-efficient manner. This work provides an in-depth overview of SNNs and provides a thorough evaluation comparison against ANNs, identifying key opportunities and challenges for the development of this promising area.
